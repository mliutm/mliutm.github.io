  • skywalking жҳҜжң¬еңҹејҖжәҗзҡ„еҹәдәҺеӯ—иҠӮз ҒжіЁе…Ҙзҡ„и°ғз”Ёй“ҫеҲҶжһҗпјҢд»ҘеҸҠеә”з”Ёзӣ‘жҺ§еҲҶжһҗе·Ҙе…·гҖӮзү№зӮ№жҳҜж”ҜжҢҒеӨҡз§ҚжҸ’д»¶пјҢUIеҠҹиғҪиҫғејәпјҢжҺҘе…Ҙз«Ҝж— д»Јз Ғдҫөе…ҘгҖӮзӣ®еүҚе·ІеҠ е…ҘApacheеӯөеҢ–еҷЁгҖӮ

Span

2.3 йӣҶжҲҗzipkin

2.3.1 zipkinд»Ӣз»Қ

Zipkin жҳҜ Twitter зҡ„дёҖдёӘејҖжәҗйЎ№зӣ®пјҢе®ғеҹәдәҺGoogle Dapperе®һзҺ°пјҢе®ғиҮҙеҠӣдәҺж”¶йӣҶжңҚеҠЎзҡ„е®ҡж—¶ж•°жҚ®пјҢд»Ҙи§ЈеҶіеҫ®жңҚеҠЎжһ¶жһ„дёӯзҡ„е»¶иҝҹй—®йўҳпјҢеҢ…жӢ¬ж•°жҚ®зҡ„ж”¶йӣҶгҖҒеӯҳеӮЁеұ•зҺ°гҖҒжҹҘжүҫе’ҢжҲ‘д»¬еҸҜд»ҘдҪҝз”Ёе®ғжқҘж”¶йӣҶеҗ„дёӘжңҚеҠЎеҷЁдёҠиҜ·жұӮй“ҫи·Ҝзҡ„и·ҹиёӘж•°жҚ®пјҢе№¶йҖҡиҝҮе®ғжҸҗдҫӣзҡ„REST APIжҺҘеҸЈжқҘиҫ…еҠ©жҲ‘д»¬жҹҘиҜўи·ҹиёӘж•°жҚ®д»Ҙе®һзҺ°еҜ№еҲҶеёғејҸзі»з»ҹзҡ„зӣ‘жҺ§зЁӢеәҸпјҢд»ҺиҖҢеҸҠж—¶ең°еҸ‘зҺ°зі»з»ҹдёӯеҮәзҺ°зҡ„е»¶иҝҹеҚҮй«ҳй—®йўҳе№¶жүҫеҮәзі»з»ҹжҖ§иғҪз“¶йўҲзҡ„ж №жәҗ

йҷӨдәҶйқўеҗ‘ејҖеҸ‘зҡ„ API жҺҘеҸЈд№ӢеӨ–пјҢе®ғд№ҹжҸҗдҫӣдәҶж–№дҫҝзҡ„UIз»„д»¶жқҘеё®еҠ©жҲ‘д»¬зӣҙи§Ӯзҡ„жҗңзҙўи·ҹиёӘдҝЎжҒҜе’ҢеҲҶжһҗиҜ·жұӮй“ҫи·ҜжҳҺз»ҶпјҢжҜ”еҰӮпјҡеҸҜд»ҘжҹҘиҜўжҹҗж®өж—¶й—ҙеҶ…еҗ„з”ЁжҲ·иҜ·жұӮзҡ„еӨ„зҗҶж—¶й—ҙзӯүгҖӮ

Zipkin жҸҗдҫӣдәҶеҸҜжҸ’жӢ”ж•°жҚ®еӯҳеӮЁж–№ејҸпјҡIn-MemoryгҖҒMySqlгҖҒCassandra д»ҘеҸҠ ElasticsearchгҖӮ

zipkinзҡ„ж ёеҝғз”ұ4йғЁеҲҶз»„жҲҗ

з¬¬дёҖз§ҚйғЁзҪІжһ¶жһ„з”ұдәҺиө„жәҗеҚ з”Ёй—®йўҳпјҢзҺ°е·ІеҫҲе°‘дҪҝз”ЁпјҢзӣ®еүҚдҪҝз”ЁжңҖеӨҡзҡ„жҳҜз¬¬дәҢз§ҚйғЁзҪІжһ¶жһ„пјҢиҮідәҺз¬¬дёүз§ҚйғЁзҪІжһ¶жһ„дёӘдәәи§үеҫ—жІЎжңүеҝ…иҰҒеј•е…Ҙж¶ҲжҒҜйҳҹеҲ—пјҢйҷӨйқһжңүе…¶д»–йңҖжұӮпјҢеӣ дёәеңЁж•°жҚ®йҮҸиҫғеӨ§зҡ„жғ…еҶөдёӢпјҢFilebeat дҪҝз”ЁеҺӢеҠӣж•Ҹж„ҹеҚҸи®®еҗ‘ Logstash жҲ– Elasticsearch еҸ‘йҖҒж•°жҚ®гҖӮ

еҰӮжһң Logstash жӯЈеңЁз№Ғеҝҷең°еӨ„зҗҶж•°жҚ®пјҢе®ғдјҡе‘ҠзҹҘ Filebeat еҮҸж…ўиҜ»еҸ–йҖҹеәҰгҖӮжӢҘеЎһи§ЈеҶіеҗҺпјҢFilebeat е°ҶжҒўеӨҚеҲқе§ӢйҖҹеәҰе№¶з»§з»ӯеҸ‘йҖҒж•°жҚ®гҖӮ
