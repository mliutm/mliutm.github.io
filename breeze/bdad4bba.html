01-еӨ„зҗҶеҷЁй—®йўҳ

|еӯ—ж•°жҖ»и®Ў:2k|йҳ…иҜ»ж—¶й•ҝ:10еҲҶй’ҹ|йҳ…иҜ»йҮҸ:

йўҳвҪ¬жҸҸиҝ°

жҹҗе…¬еҸёз ”еҸ‘дәҶвјҖж¬ҫвҫјжҖ§иғҪAIеӨ„зҗҶеҷЁгҖӮжҜҸеҸ°зү©зҗҶи®ҫеӨҮе…·еӨҮ8йў—AIеӨ„зҗҶеҷЁпјҢзј–еҸ·еҲҶеҲ«дёә0гҖҒ1гҖҒ2гҖҒ3гҖҒ4гҖҒ

5гҖҒ6гҖҒ7гҖӮ

зј–еҸ·0-3зҡ„еӨ„зҗҶеҷЁеӨ„дәҺеҗҢвјҖдёӘй“ҫи·ҜдёӯпјҢзј–еҸ·4-7зҡ„еӨ„зҗҶеҷЁеӨ„дәҺеҸҰеӨ–вјҖдёӘй“ҫи·ҜдёӯпјҢдёҚйҖҡй“ҫи·Ҝдёӯзҡ„еӨ„зҗҶеҷЁ

дёҚиғҪйҖҡдҝЎгҖӮ

еҰӮдёӢеӣҫжүҖзӨәгҖӮзҺ°з»ҷе®ҡжңҚеҠЎеҷЁеҸҜвҪӨзҡ„еӨ„зҗҶеҷЁзј–еҸ·ж•°з»„arrayпјҢд»ҘеҸҠд»»еҠЎз”іиҜ·зҡ„еӨ„зҗҶеҷЁж•°йҮҸnumпјҢжүҫеҮәз¬Ұ

еҗҲдёӢеҲ—дәІе’ҢжҖ§и°ғеәҰеҺҹеҲҷзҡ„иҠҜвҪҡз»„еҗҲгҖӮ

еҰӮжһңдёҚеӯҳеңЁз¬ҰеҗҲиҰҒжұӮзҡ„з»„еҗҲпјҢеҲҷиҝ”еӣһз©әеҲ—иЎЁгҖӮ

дәІе’ҢжҖ§и°ғеәҰеҺҹеҲҷпјҡ

-еҰӮжһңз”іиҜ·еӨ„зҗҶеҷЁдёӘж•°дёә1пјҢеҲҷйҖүжӢ©еҗҢвјҖй“ҫи·ҜпјҢеү©дҪҷеҸҜвҪӨзҡ„еӨ„зҗҶеҷЁж•°йҮҸдёә1дёӘзҡ„жңҖдҪіпјҢе…¶ж¬ЎжҳҜеү©дҪҷ3дёӘ

зҡ„дёәж¬ЎдҪіпјҢз„¶еҗҺжҳҜеү©дҪҷ2дёӘпјҢжңҖеҗҺжҳҜеү©дҪҷ4дёӘгҖӮ

-еҰӮжһңз”іиҜ·еӨ„зҗҶеҷЁдёӘж•°дёә2пјҢеҲҷйҖүжӢ©еҗҢвјҖй“ҫи·Ҝеү©дҪҷеҸҜвҪӨзҡ„еӨ„зҗҶеҷЁж•°йҮҸ2дёӘзҡ„дёәжңҖдҪіпјҢе…¶ж¬ЎжҳҜеү©дҪҷ4дёӘпјҢ

жңҖеҗҺжҳҜеү©дҪҷ3дёӘгҖӮ

-еҰӮжһңз”іиҜ·еӨ„зҗҶеҷЁдёӘж•°дёә4пјҢеҲҷеҝ…йЎ»йҖүжӢ©еҗҢвјҖй“ҫи·Ҝеү©дҪҷеҸҜвҪӨзҡ„еӨ„зҗҶеҷЁж•°йҮҸдёә4дёӘгҖӮ

-еҰӮжһңз”іиҜ·еӨ„зҗҶеҷЁдёӘж•°дёә8пјҢеҲҷз”іиҜ·иҠӮзӮ№жүҖжңү8дёӘеӨ„зҗҶеҷЁгҖӮ

жҸҗзӨәпјҡ

  1. д»»еҠЎз”іиҜ·зҡ„еӨ„зҗҶеҷЁж•°йҮҸеҸӘиғҪжҳҜ1гҖҒ2гҖҒ4гҖҒ8гҖӮ

  2. зј–еҸ·0-3зҡ„еӨ„зҗҶеҷЁеӨ„дәҺвјҖдёӘй“ҫи·ҜпјҢзј–еҸ·4-7зҡ„еӨ„зҗҶеҷЁеӨ„дәҺеҸҰеӨ–вјҖдёӘй“ҫи·ҜгҖӮ

  3. еӨ„зҗҶеҷЁзј–еҸ·е”ҜвјҖпјҢдё”дёҚеӯҳеңЁзӣёеҗҢзј–еҸ·еӨ„зҗҶеҷЁгҖӮ

иҫ“вјҠжҸҸиҝ°

иҫ“вјҠеҢ…еҗ«еҸҜвҪӨзҡ„еӨ„зҗҶеҷЁзј–еҸ·ж•°з»„arrayпјҢд»ҘеҸҠд»»еҠЎз”іиҜ·зҡ„еӨ„зҗҶеҷЁж•°йҮҸnumдёӨдёӘйғЁеҲҶгҖӮ

з¬¬вјҖвҫҸдёәarrayпјҢз¬¬вјҶвҫҸдёәnumгҖӮдҫӢеҰӮпјҡ

[0, 1, 4, 5, 6, 7]

1

иЎЁзӨәеҪ“еүҚзј–еҸ·дёә0гҖҒ1гҖҒ4гҖҒ5гҖҒ6гҖҒ7зҡ„еӨ„зҗҶеҷЁеҸҜвҪӨгҖӮд»»еҠЎз”іиҜ·1дёӘеӨ„зҗҶеҷЁгҖӮ

  • 0 <= array.length <= 8
  • 0 <= array[i] <= 7
  • num in [1, 2, 4, 8]

иҫ“еҮәжҸҸиҝ°

иҫ“еҮәдёәз»„еҗҲеҲ—иЎЁпјҢеҪ“array=[0пјҢ1пјҢ4пјҢ5пјҢ6пјҢ7]пјҢnum=1 ж—¶пјҢиҫ“еҮәдёә[[0], [1]]гҖӮ

вҪӨдҫӢ

иҫ“е…Ҙ[0, 1, 4, 5, 6, 7]
1
иҫ“еҮә[[0], [1]]
иҜҙжҳҺж №жҚ®з¬¬вјҖжқЎдәІе’ҢжҖ§и°ғеәҰеҺҹеҲҷпјҢеңЁеү©дҪҷдёӨдёӘеӨ„зҗҶеҷЁзҡ„й“ҫи·ҜпјҲ0, 1, 2, 3пјүдёӯйҖүжӢ©еӨ„зҗҶеҷЁгҖӮз”ұдәҺеҸӘжңү0е’Ң1еҸҜвҪӨпјҢеҲҷиҝ”еӣһд»»ж„ҸвјҖйў—еӨ„зҗҶеҷЁеҚіеҸҜгҖӮ
иҫ“е…Ҙ[0, 1, 4, 5, 6, 7]
4
иҫ“еҮә[[4, 5, 6, 7]]
иҜҙжҳҺж №жҚ®з¬¬дёүжқЎдәІе’ҢжҖ§и°ғеәҰеҺҹеҲҷпјҢеҝ…йЎ»йҖүжӢ©еҗҢвјҖй“ҫи·Ҝеү©дҪҷеҸҜвҪӨзҡ„еӨ„зҗҶеҷЁж•°йҮҸдёә4дёӘзҡ„зҺҜ

йўҳвҪ¬и§Јжһҗ

вҪӨдҫӢдёӯпјҢй“ҫи·Ҝlink1=[0,1]пјҢй“ҫи·Ҝlink2=[4,5,6,7]

зҺ°еңЁиҰҒйҖү1дёӘеӨ„зҗҶеҷЁпјҢеҲҷйңҖиҰҒжҢүз…§дәІе’ҢжҖ§и°ғеәҰеҺҹеҲҷ

еҰӮжһңз”іиҜ·еӨ„зҗҶеҷЁдёӘж•°дёә1пјҢеҲҷйҖүжӢ©еҗҢвјҖй“ҫи·ҜпјҢеү©дҪҷеҸҜвҪӨзҡ„еӨ„зҗҶеҷЁж•°йҮҸдёә1дёӘзҡ„жңҖдҪіпјҢе…¶ж¬ЎжҳҜеү©дҪҷ

3дёӘзҡ„дёәж¬ЎдҪіпјҢз„¶еҗҺжҳҜеү©дҪҷ2дёӘпјҢжңҖеҗҺжҳҜеү©дҪҷ4дёӘгҖӮ

жңҖдҪізҡ„жҳҜпјҢжүҫеү©дҪҷеҸҜвҪӨ1дёӘеӨ„зҗҶеҷЁзҡ„й“ҫи·ҜпјҢеҸ‘зҺ°жІЎжңүпјҢlink1еү©дҪҷеҸҜвҪӨ2пјҢlink2еү©дҪҷеҸҜвҪӨ4

е…¶ж¬Ўзҡ„жҳҜпјҢжүҫеү©дҪҷеҸҜвҪӨ3дёӘеӨ„зҗҶеҷЁзҡ„й“ҫи·ҜпјҢеҸ‘зҺ°жІЎжңү

еҶҚж¬Ўзҡ„жҳҜпјҢжүҫеү©дҪҷеҸҜвҪӨ2дёӘеӨ„зҗҶеҷЁзҡ„й“ҫи·ҜпјҢlink1з¬ҰеҗҲиҰҒжұӮпјҢеҚід»Һ0е’Ң1еӨ„зҗҶеҷЁдёӯд»»йҖүвјҖдёӘпјҢжңүдёӨз§ҚйҖү
